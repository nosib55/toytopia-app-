# Toytopia

## Project Purpose
Toytopia is an online toy store for kids, designed to provide a fun and interactive shopping experience. Users can browse a wide range of toys, view detailed information, and make purchases easily. The project focuses on responsive design, smooth navigation, and user-friendly features.

## Live URL
You can access the live project here: [https://toy-topiabdx.netlify.app/](https://toy-topiabdx.netlify.app/)

## Key Features
- User authentication with Google and Email/Password
- Password reset
- View detailed toy information
- Responsive design for mobile and desktop devices
- Toast notifications for actions like login, signup, and errors
- Interactive UI with icons and animations

## NPM Packages Used
- `react` – For building the user interface
- `react` – For rendering React components
- `react-router` – For handling client-side routing
- `firebase` – For authentication and backend services
- `react-icons` – For adding icons
- `react-toastify` – For toast notifications
- `tailwindcss` – For styling
